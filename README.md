# NKUST 行事曆轉換工具

將高科大學期行事曆 PDF，一鍵轉換為可匯入的數位行事曆。

---

##  功能特色

-  上傳高科大行事曆 PDF
-  自動解析事件與日期
-  可選擇需要的事件
-  匯出 `.ics` 行事曆檔案
-  支援 Google Calendar / Apple Calendar / Outlook

---

##  工具連結

👉 https://nkust-cal-convert.vercel.app/

---

##  使用流程

1. 進入高科大教務系統下載學期行事曆 PDF  
2. 上傳 PDF 至本網站  
3. 系統自動解析事件內容與時間  
4. 勾選欲加入的行事曆事件  
5. 匯出 `.ics` 檔案並匯入個人行事曆

**初次使用載入和轉換時間較長，請使用者耐心等候～**

## 使用技術
- 前端：Next.js / React / TailwindCSS  
- 後端：Golang  
- PDF 處理：自訂解析邏輯  
- 輸出格式：ICS（iCalendar）
